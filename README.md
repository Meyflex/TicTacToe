## Usage

* 2 players. the first click is for player1, the second for player2, the third for player1...
* board is 3x3
* each time a player win (your code detects this case), you will increase his score and erase the
board to start a new match.
* display a clock on the top of the screen (MM:SS), countdown from 03:00 to 00:00, with a
refresh every second. at 00:00, a pop-up display on screen the winner and the score (Winner:
Player X 5/2!)
